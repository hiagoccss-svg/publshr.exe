Publshr — macOS native desktop installer
========================================

Download the latest installer zip from GitHub Releases (tag: live):
  Publshr-Install-macos.zip

Or use the one-line install (no zip needed):
  curl -fsSL "https://raw.githubusercontent.com/hiagoccss-svg/publshr.exe/refs/heads/main/install-macos.sh" | bash

OPTION A — Double-click
  1. Unzip Publshr-Install-macos.zip
  2. Double-click "Publshr Install.command"
  3. Complete the native Installer or Terminal prompts

OPTION B — From this folder (git clone)
  bash "../../install-macos.sh"
  — or double-click "Publshr Install.command" here

REQUIREMENTS: macOS 14+, Apple Silicon or Intel, internet, admin password once.

Installed app: /Applications/Publshr.app
Auto-updates: yes (push to main → live channel)
