# macOS (legacy path)

The Mac and Linux native CLI has moved to **[`native/publshr`](../native/publshr)**.

Use from the repo root:

```bash
make -C .. install-local   # install into this repo's .local/
./install-local.sh
```

The old `mac/publshr` tree may remain until removed; **`native/publshr` is canonical.**
