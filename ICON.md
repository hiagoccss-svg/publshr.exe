# Brand icon

**`icon.png` in this folder is the only source of truth** for the macOS app icon, installer, and in-app logo.

Replace this file, then push to `main` (or run `bash mac/publshr/scripts/sync-app-icon.sh` locally). Builds normalize it onto a **full white** background for Dock/Finder and UI.

Details: [mac/publshr/app/ICON.md](mac/publshr/app/ICON.md)
